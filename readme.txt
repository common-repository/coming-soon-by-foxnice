Super simple and easy lightweight coming soon plugin.

== Description ==

Coming Soon by Foxnice is a super lightweight and simple plugin that allows you to easily activate or deactivate a "Coming Soon" page for your website. It’s ideal for use while your site is under development or maintenance. With a size of just 98KB, this plugin doesn’t unnecessarily load or slow down your website, offering a basic but effective solution for these scenarios. Perfect for quickly setting up a "Coming Soon" page without any frills.

== Installation ==

1 - Upload the entire coming-soon-foxnice folder to the /wp-content/plugins/ directory.
2 - Activate the plugin through the Plugins screen (Plugins > Installed Plugins).

You will find the Coming Soon option in your WordPress admin screen under Settings.

For basic usage and setup instructions, check the plugin's website.
